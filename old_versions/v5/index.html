Architecture

How it works

PDF is read client-side via FileReader. A Vercel serverless function appends the API key and proxies to the LLM. Structured JSON comes back. No file storage, no auth required for free tier.

Features

What the app does

📄
Multi-source synthesis
Feed in PDFs from different sources. Get the overlapping facts and key themes — without the editorial framing of any single one.
🎯
6 domain templates
General, Financial, Legal, Academic, Medical, Executive. Auto-detected from document content.
📤
Export
One-tap .docx and .pptx. Generated client-side, no server round-trip.
🔀
Version diff
Two versions of a document in, structured comparison out. Change significance rated per item.
🔒
No file storage
Read via FileReader, sent to LLM, never stored. Briefings saved on-device only.
📱
WhatsApp delivery
Premium: forward a PDF via WhatsApp, get summary + .docx + .pptx back on your phone.
