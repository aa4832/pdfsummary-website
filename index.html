What's different

The work happens
before and after
the API call.

Anyone can send a PDF to an LLM. We built what the API can't do — document comparison, persistent search, domain-specific structure, and professional export.

🔀

Document comparisonNew

Upload two versions of a contract, quarterly report, or policy document. See exactly what changed — added clauses, removed terms, shifted numbers — in a structured diff view.

📚

Analysis libraryNew

Every analysis is saved locally and searchable. Find last quarter's earnings analysis, compare it with this quarter's, track how metrics changed over time. Your documents build on each other.

🔍

Auto-detection

The app reads the first pages and automatically selects the right analysis template — financial, legal, medical, academic, or executive. You can override, but you shouldn't need to.

📐

Structured export

Not a text dump. Word exports use proper headings, styled tables, and consistent formatting. PowerPoint exports create actual slide decks with themed layouts and data cards.

How it works

Three steps.

Select PDFs or text files. The app auto-detects the document type and suggests the right analysis template. Override if you want.

Analyze

Claude or Gemini reads every page with a domain-specific prompt. Large files are chunked automatically. Results are saved to your library.

03

Export

Structured briefing with proper formatting. Export as Word (.docx) or PowerPoint (.pptx). Compare with previous analyses from your library.
